  БОТ ВОЙНЫ КЛАНОВ
============================

УСТАНОВКА (Pydroid3):
---------------------
1. Открой терминал в Pydroid3
2. Установи зависимости:
   pip install aiogram==3.7.0 aiosqlite==0.20.0

3. Открой файл config.py и замени:
   BOT_TOKEN = "ВСТАВЬ_ТОКЕН_СЮДА"   → токен от @BotFather
   ADMIN_IDS = [123456789]            → твой ID от @userinfobot

4. Добавь бота в свою группу и дай права АДМИНИСТРАТОРА

5. Напиши /start в группе — бот запомнит чат

6. Запусти main.py в Pydroid3

СТРУКТУРА ФАЙЛОВ:
-----------------
main.py           — точка входа
config.py         — все настройки
requirements.txt  — зависимости

db/
  database.py     — инициализация БД
  clan_queries.py — запросы по кланам
  duel_queries.py — запросы по дуэлям

game/
  mines.py        — логика игры мины
  matchmaking.py  — подбор пар

handlers/
  clan.py         — создание/вступление/профиль кланов
  duel.py         — игровой процесс дуэли
  scheduler.py    — автоматические дуэли каждые 48ч

КОМАНДЫ:
--------
/createclan     — создать клан
/join           — вступить в клан (листалка)
/clan           — профиль клана
/top            — таблица всех кланов
/leaveclan      — выйти из клана
/kick           — кикнуть участника (только лидер, ответом)
/transferlead   — передать лидерство (только лидер, ответом)
/minduel        — начать игру когда тебя вызвали на дуэль

КАК РАБОТАЮТ ДУЭЛИ:
-------------------
- Каждые 48 часов бот сам выбирает 2 игрока из разных кланов
- Аутсайдер (меньше Al) vs лидер (больше Al)
- Игроки пишут /minduel боту в личку
- Выбирают количество мин (1-6) → открывают поле 5x5
- При мине: Al клана × 0.75
- При выигрыше: Al клана × выбранный множитель
- Война длится 30 дней, потом сброс

НАСТРОЙКИ (config.py):
----------------------
WAR_DURATION_DAYS  — длительность войны (по умолч. 30)
STARTING_AL        — стартовые очки (по умолч. 100)
DUEL_INTERVAL_HOURS — интервал дуэлей (по умолч. 48)
LOSE_MULTIPLIER    — штраф за мину (по умолч. 0.75)
